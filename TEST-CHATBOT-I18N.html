🧪 How to Test

Step 1: Open Portfolio

Open D:\BlogAdmin\index.html in your browser (Chrome/Edge recommended)

Open Portfolio

Step 2: Test Language Switching

  1. Click the 🤖 chatbot button (bottom-right corner)
  2. Observe the chatbot window opens with Vietnamese text
  3. Click VN button (top-right) - should stay Vietnamese
  4. Click EN button - all chatbot text should change to English
  5. Click 中 button - all chatbot text should change to Chinese
  6. Check that the input placeholder also changes

Step 3: Test Quick Actions

  1. Select Vietnamese (VN)
  2. Click "💼 Kinh nghiệm" button
  3. Check console (F12) - should see: "Kinh nghiệm làm việc của bạn là gì?"
  4. Switch to English (EN)
  5. Click "💼 Experience" button
  6. Check console - should see: "What is your work experience?"

Step 4: Check Console for Errors

  1. Press F12 to open Developer Tools
  2. Go to Console tab
  3. Switch between languages multiple times
  4. You should see: ✅ No errors, only info messages

⚠️ API Key Required

To get API responses working:

  1. Get free API key: https://makersuite.google.com/app/apikey
  2. Open D:\BlogAdmin\chatbot.js
  3. Replace line 6:
