## 📞 How to Update Content

### Add New AI Project
Edit `index.html`, find `.ai-project-cards` section:
```html
<div class="ai-project-card">
    <div class="ai-project-header">
        <div class="ai-project-icon">🎯</div>
        <div>
            <h4>Your Project Name</h4>
            <p class="project-subtitle">Subtitle</p>
        </div>
    </div>
    <!-- Add content here -->
</div>
```

### Add Research Topic
Find `.research-grid` section:
```html
<div class="research-card">
    <div class="research-header">
        <span class="research-icon">🔬</span>
        <h4>Topic Name</h4>
    </div>
    <ul class="research-list">
        <li>Point 1</li>
        <li>Point 2</li>
    </ul>
    <div class="research-status">Status</div>
</div>
```

### Add Timeline Item
Find `.learning-timeline`:
```html
<div class="learning-item completed">
    <div class="learning-marker"></div>
    <div class="learning-content">
        <h4>Title</h4>
        <p>Description</p>
        <span class="learning-date">Date</span>
    </div>
</div>
```

**Status classes:** `completed`, `in-progress`, `planned`

## 🎨 Customization

### Change Colors
Edit `enhancements.css`:
```css
.ai-focus-card {
    border-color: #your-color;
}
```

### Change Icons
Replace emoji icons in HTML:
```html
<div class="ai-focus-icon">🤖</div>  <!-- Change this -->
```

### Adjust Animations
Edit animation duration:
```css
.ai-focus-icon {
    animation: float 3s ease-in-out infinite; /* Change 3s */
}
```
